🔥 Firebase Setup — Safety Director

Connect your training portal to Firebase so every quiz result is automatically recorded in a central database. Do this once — then share a link with your foremen and you're done.

Firebase is not yet configured on this device.
1
Create a Free Firebase Project
▼

1. Go to firebase.google.com and sign in with your Google account.

2. Click "Go to console", then "Add project".

3. Name it BBB-Safety-Training (or anything you like). Click Continue.

4. On the Google Analytics screen — you can disable it, it's not needed. Click "Create project".

5. Wait about 30 seconds for the project to be created, then click "Continue".

2
Add a Web App & Get Your Config
▼

1. In your Firebase project dashboard, click the </> (Web) icon to add a web app.

2. Give it a nickname like BBB Training Portal. Click "Register app".

3. Firebase will show you a firebaseConfig block. It looks like this:

const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "bbb-safety-training.firebaseapp.com", projectId: "bbb-safety-training", storageBucket: "bbb-safety-training.appspot.com", messagingSenderId: "123456789", appId: "1:123456789:web:abc123def456" };

4. Copy that entire block. You'll paste it below in Step 4.

5. Click "Continue to console".

3
Create the Firestore Database
▼

1. In the left sidebar, click Build → Firestore Database.

2. Click "Create database".

3. Select "Start in test mode" (we'll set proper rules in the next step). Click Next.

4. Choose a Cloud Firestore location closest to you — us-central is fine. Click "Enable".

5. Wait about 30 seconds for the database to be ready.

4
Set Security Rules
▼

1. In Firestore, click the "Rules" tab at the top.

2. Replace everything in the rules editor with the following:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /training_records/{docId} { // Anyone can submit a training record allow create: if true; // Only allow reading (for director dashboard) allow read: if true; // Nobody can modify or delete records (audit integrity) allow update, delete: if false; } } }

3. Click "Publish".

Why these rules? Employees can only ADD records (not read or change others' data). The director dashboard can read all records. Nobody can delete or alter submitted records — protecting your audit trail.
5
Enter Your Firebase Config Below
▼

Paste your firebaseConfig object from Step 2 into the box below, then click Save & Connect.
